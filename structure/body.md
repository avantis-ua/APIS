# Тело ответа `body`
```json
"body": {
    "items": [
        {
            "item": {
                "resource_id": "",
                "resource_field": ""
            }
        }
    ]
}
```
В будущем рассматривается появления таких параметров как например: 
- `body`/`memu_items`/`menu_item`/`{arr}`
- `body`/`categories`/`category`/`{arr}`
- и других
