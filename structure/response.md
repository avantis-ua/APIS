# Ответ `response`
```json
{
    "response": {
        "total": "100",
        "id": "10",
        "resource_param": "",
    }
}
```
### total
`total` - Колличество выведенных записей. Обязательный параметр !
### resource_param
Дополнительно можно отдавать любую информацию разрешенную данным ресурсом.
